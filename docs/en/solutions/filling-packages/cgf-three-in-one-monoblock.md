# Rinsing, filling and sealing three-line machine

![Three-in-one rinsing, filling and sealing machine](/images/filling-products/cgf-three-in-one-monoblock.webp)

The three-line rinsing, filling, and sealing machine is one of the main models in the combined machine and filling supporting bus. It is suitable for rinsing, filling, and capping to be completed on one piece of equipment, reducing intermediate transfers, shortening the floor space, and improving the efficiency of the entire line.

## Model positioning

- Series: [Combined machine and filling supporting bus](/en/solutions/filling-packages)
- Parameter overview entrance: [Filling and packaging equipment parameter library](/en/products/filling-equipment#fp-cgf)
- Applicable scenarios: Integrated packaging line suitable for glass bottles or rigid bottles such as bottled oil, liquor, seasoning liquid, etc.

## Summary of public information

- A hot-selling model that integrates rinsing, filling and capping.
- When the calibrated capacity is `500 ml`, the reference filling accuracy can reach `±1.5 ml`.
- It can realize continuous actions such as starting with bottles, stopping without bottles, automatic rinsing, filling, sealing and so on.

## Model Overview

- `CGF30-24-8`: suitable for medium-sized production lines, reference output `6000 p/h`
- `CGF40-30-8`: The standard line suitable for continued exploration of production capacity, reference output `8000 p/h`
- `CGF48-36-8`: suitable for medium to high output continuous production, reference output `10000 p/h`
- `CGF60-40-8`: suitable for higher-tempo whole line projects, reference output `12000 p/h`
