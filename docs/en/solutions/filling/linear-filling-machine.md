# Linear filling machine

![Linear filling machine](/images/filling-products/linear-filling-machine.webp)

The linear filling machine is mainly used for quantitative filling of medium and large barrels of liquids, suitable for packaging needs in the 10-25 L capacity segment, and is commonly used in edible oil, chemical liquids and large packaging liquid projects.

## Model positioning

- Series: [Filling Machine Series](/en/solutions/filling/)
- Parameter overview entrance: [Filling and packaging equipment parameter library](/en/products/filling-equipment#fill-linear)
- Applicable scenarios: Suitable for linear filling of medium and large barrels and large-capacity packaging projects.

## Summary of public information

- Public model coverage `ZGP-4 / ZGP-6 / ZGP-8 / ZGP-10`
- Filling range `10-25 L`
- Reference accuracy `≤0.2%`

## How to choose

- `ZGP-4 / ZGP-6`: suitable for medium and low production large barrel projects
- `ZGP-8`: suitable for medium-sized main lines
- `ZGP-10`: Suitable for continuous filling of vats with higher cycle times
