# GFP-48H low vacuum filling machine

![GFP-48H low vacuum filling machine](/images/filling-products/gfp-48h-low-vacuum-filling-machine.webp)

`GFP-48H` is a high-capacity model in the low-vacuum constant-level filling series, suitable for 15,000 bottles/hour wine, rice wine or glass bottle flavoring liquid projects.

## Core parameters

| Project | Parameters |
| --- | --- |
| Model | GFP-48H |
| Number of filling heads | 48 |
| Reference production | 15,000 bottles/hour |
| Total power | 6 kW |
| Overall dimensions | 2800×2900×2200 mm |
| Filling method | Low vacuum/constant level filling |
| Applicable liquids | Wine, rice wine, soy sauce, vinegar and other easily foaming liquids |

## What does this device do?

GFP-48H is suitable for continuous filling scenarios with higher tempo, focusing on balancing liquid level stability, foam control and higher conveying tempo. If automatic cap shrinking, labeling, cartoning and palletizing have been planned for the later part of the project, this level will better match the entire line's capabilities.

## Public Features

- Low vacuum liquid level filling, suitable for liquids that are prone to foaming.
- The liquid flows down the bottle wall, which can reduce foaming during filling.
- Reliable structure, suitable for high-tempo production.
- The bottle support is gentle, which can reduce the risk of bottle damage.
- No filling without bottles, improving the stability of the back-end line.
- Easy to disassemble, wash and maintain, suitable for large-scale production.

## Which projects are suitable for

- A high-tempo wine project with a production capacity of around 15,000 bottles/hour.
- A complete glass bottle line with relatively complete post-packaging automation configuration.
- Customers who need to compromise between liquid level appearance and production capacity.
