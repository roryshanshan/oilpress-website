# 柴火蒸锅
![柴火蒸锅](https://i.postimg.cc/Gcs3LvWX/image.png?dl=1)
## 产品概述

灶体采用Q355B材质打造，可使用不同燃料加热。
灶体内部采用集热式管道，增大受热面积，节省燃料。
集成式水箱箱体，靠近灶体底部下设排水口。
锅体采用304不锈钢材质。
