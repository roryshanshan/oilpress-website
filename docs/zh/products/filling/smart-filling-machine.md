# 系列智能型灌装机

![系列智能型灌装机](/images/filling-products/smart-filling-machine.webp)

系列智能型灌装机主要面向零售瓶装液体自动线，兼顾灌装精度、速度和多瓶型适配，适合食用油、调味液等常压灌装场景做中高节拍生产。

## 机型定位

- 所属系列：[灌装机系列](/zh/solutions/filling/)

- 适用场景：适合作为零售瓶装油、调味液主灌装机。

## 怎么选

- `12-18 头`：更适合起步型或中型瓶装项目
- `24-30 头`：适合常见主力产线，兼顾速度与占地
- `36 头`：适合更高节拍的连续化自动线
