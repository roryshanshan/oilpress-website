# 直线式灌装机

![直线式灌装机](/images/filling-products/linear-filling-machine.webp)

直线式灌装机主要面向中大桶液体的定量灌装，适合 10-25 L 容量段的包装需求，常用于食用油、化工液体和大包装液体项目。
