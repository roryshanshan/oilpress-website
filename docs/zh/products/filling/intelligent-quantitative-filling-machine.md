# 智能定量灌装机

![智能定量灌装机](/images/filling-products/intelligent-quantitative-filling-machine.webp)

智能定量灌装机强调灌装与后续理盖、扣盖、封盖动作的联动，一台设备即可覆盖多道动作，适合希望减少中间人工干预和二次污染风险的自动化线体。

## 机型定位

- 所属系列：[灌装机系列](/zh/solutions/filling/)
- 参数总览入口：[灌装包装设备参数库](/zh/products/filling-equipment#fill-dk)
- 适用场景：适合灌装后联动提盖、理盖、扣盖、封盖的自动线。

## 公开资料摘要

- 结构紧凑，适合自动化升级项目
- 覆盖 `DK-12` 到 `DK-36` 机型
- 参考效率 `4000-13000 瓶/小时`
