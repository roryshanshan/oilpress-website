# 包饼机
![包饼机](https://i.postimg.cc/nFBLkqTX/image.png?dl=1)
## 产品概述

该机器为卧式液压榨油机必备配套设备，压饼模具大小可定制，常规款饼径240-480mm，其他款尺寸需定制。
因生产批次不同，机器外观、颜色、尺寸、重量、材质会有差异，后期不另做通知。
