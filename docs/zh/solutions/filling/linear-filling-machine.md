# 直线式灌装机

![直线式灌装机](/images/filling-products/linear-filling-machine.webp)

直线式灌装机主要面向中大桶液体的定量灌装，适合 10-25 L 容量段的包装需求，常用于食用油、化工液体和大包装液体项目。

## 机型定位

- 所属系列：[灌装机系列](/zh/solutions/filling/)
- 参数总览入口：[灌装包装设备参数库](/zh/products/filling-equipment#fill-linear)
- 适用场景：适合中大桶直线灌装和大容量包装项目。

## 公开资料摘要

- 公开机型覆盖 `ZGP-4 / ZGP-6 / ZGP-8 / ZGP-10`
- 灌装范围 `10-25 L`
- 参考精度 `≤0.2%`

## 怎么选

- `ZGP-4 / ZGP-6`：适合中低产量大桶项目
- `ZGP-8`：适合中型主力线
- `ZGP-10`：适合更高节拍的大桶连续灌装
