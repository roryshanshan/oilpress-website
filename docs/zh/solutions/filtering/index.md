# 过滤机系列

![气动千层滤油机实拍](/images/filling-solutions/filtering.png)

过滤机系列主要用于液体产品进入灌装线前的澄清、精滤和杂质控制。对于食用油项目，过滤不仅关系到油品外观和沉淀控制，也会影响灌装稳定性、喷码清晰度和成品货架表现。

## 主要作用

- 去除悬浮杂质、细小颗粒和部分沉淀
- 提升成品清亮度与外观稳定性
- 减少灌装头、阀门和管路堵塞风险
- 为后续包装、精炼或高端产品销售提供更稳定油品状态

## 适用介质

- 花生油、菜籽油、大豆油、芝麻油等食用油
- 葡萄酒、果酒、饮料、调味液
- 对澄清度和稳定性要求较高的液体产品

## 选型建议

- 先确认介质黏度、杂质量和目标过滤精度
- 再确认是作为压榨后澄清过滤，还是灌装前精细过滤
- 若为食用油项目，建议结合[过滤设备](/zh/products/filtration-equipment)与[气动千层滤油机](/zh/products/pneumatic-filter-press)一起判断
- 若过滤后直接包装，可继续联动[灌装机系列](/zh/solutions/filling/)

## 典型配套关系

1. 原液储罐或缓冲罐
2. 过滤机系列
3. [灌装机系列](/zh/solutions/filling/)
4. [封口机系列](/zh/solutions/sealing/)
