# 冲瓶、灌装、封口三联机

![冲瓶、灌装、封口三联机](/images/filling-products/cgf-three-in-one-monoblock.webp)

冲瓶、灌装、封口三联机是联体机与灌装配套总线中的主力机型之一，适合把冲瓶、灌装、封盖集中在一台设备上完成，减少中间转运、缩短占地长度，并提升整线联动效率。

## 机型定位

- 所属系列：[联体机与灌装配套总线](/zh/solutions/filling-packages)
- 参数总览入口：[灌装包装设备参数库](/zh/products/filling-equipment#fp-cgf)
- 适用场景：适合瓶装油、白酒、调味液等玻璃瓶或硬质瓶的一体化包装线。

## 公开资料摘要

- 热销机型，集冲瓶、灌装、封盖为一体。
- 标定容量 `500 ml` 时，参考灌装精度可达 `±1.5 ml`。
- 可实现来瓶启动、无瓶停机、自动冲瓶、灌装、封口等连续动作。

## 型号快览

- `CGF30-24-8`：适合中型产线，参考产量 `6000 p/h`
- `CGF40-30-8`：适合产能继续上探的标准线，参考产量 `8000 p/h`
- `CGF48-36-8`：适合中高产量连续生产，参考产量 `10000 p/h`
- `CGF60-40-8`：适合更高节拍整线项目，参考产量 `12000 p/h`
