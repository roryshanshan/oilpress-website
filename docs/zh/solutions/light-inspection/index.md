# 灯检机系列

![灯检机设备实拍](/images/filling-solutions/light-inspection.webp)

灯检机位于灌装和封口之后，是包装线中非常重要的质量控制节点。对于瓶装食用油、果酒、调味液等产品，灯检设备可帮助企业及时发现液位异常、异物、瓶体缺陷、封口不良等问题，减少不良品流入后段贴标和装箱工位。

## 主要检测内容

- 液位是否达标
- 瓶内是否存在悬浮物或可见异物
- 封口、旋盖或打塞是否到位
- 瓶体破损、瓶口异常、瓶盖歪斜等外观问题

## 适用场景

- 透明瓶或半透明瓶包装的食用油产品
- 高附加值小瓶油、礼盒油和出口包装项目
- 需要提升质检稳定性和减少人工目检负担的生产线
- 饮料、葡萄酒、酱油、醋等液体包装行业

## 选型建议

- 低速线可采用人工灯检平台或简化检测工位
- 中高速线建议采用自动灯检或视觉检测方案
- 如果瓶型较多，需确认设备调节范围和换型效率
- 若后段还需贴标和装箱，建议将灯检放在[封口机系列](/zh/solutions/sealing/)之后、[贴标机系列](/zh/solutions/labeling/)之前
