# GFP-12B разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-12B](/images/filling-products/gfp-12b-low-vacuum-filling-machine.webp)

«GFP-12B» — это модель с 12 головками из серии для наполнения с постоянным уровнем низкого вакуума. Публичная информация показывает, что он находится в том же сегменте производственной мощности, что и GFP-12A, и подходит для розлива небольших и средних партий вина, рисового вина, соевого соуса, уксуса и других легко пенящихся жидкостей.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-12B |
| Количество заправочных головок | 12 |
| Эталонный выход | 3000 бутылок/час |
| Общая мощность | 1,12 кВт |
| Размеры | 1045×1050×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-12B по-прежнему представляет собой решение для розлива жидкости при низком вакууме, ориентированное на контроль пенообразования, стабилизацию уровня жидкости и сокращение потерь в бутылках. Если ваш объект представляет собой линию по производству спиртных напитков малой или средней производительности или линию по производству приправ, и вы хотите сохранить место для последующего закупоривания, усадки крышек и расширения маркировки, этот тип модели с 12 головками обычно является лучшим выбором.

## Публичные функции

- Наполнение по уровню жидкости, подходит для безвоздушных, но склонных к пенообразованию жидкостей.
- Жидкость стекает по стенкам бутылки, помогая подавить пенообразование во время наполнения.
- Компактная конструкция, подходящая для размещения цехов малых и средних производственных мощностей.
- Эластичная опорная конструкция для бутылок может уменьшить потери стеклянных бутылок в процессе наполнения.
- Отсутствие розлива без бутылок помогает снизить вероятность пустого розлива и попадания неквалифицированных бутылок в последующий процесс.
- Заправочный клапан относительно легко разобрать и очистить, он подходит для повседневной очистки вина и ароматизирующих жидкостей.

## Для каких проектов подходят

- Проект по розливу спиртных напитков с производственной мощностью около 3000 бутылок/час.
- Наполнение стеклянных бутылок соевым соусом, уксусом и другими легко пенящимися жидкостями.
- Малые и средние клиенты, которым необходимо сначала установить одну машину, а затем расширить ее до полной линии.
