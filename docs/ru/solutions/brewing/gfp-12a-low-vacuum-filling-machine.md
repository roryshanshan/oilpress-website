# GFP-12A разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-12A](/images/filling-products/gfp-12a-low-vacuum-filling-machine.webp)

«GFP-12A» — это модель с 12 головками из серии для наполнения с постоянным уровнем низкого вакуума. В основном используется для наполнения стеклянных бутылок вина, рисового вина, соевого соуса, уксуса и других легко пенящихся жидкостей. Он больше подходит для проектов малых и средних производственных мощностей, пробных производственных линий на винодельнях и автономных конфигураций, требующих стабильного уровня жидкости.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | ГФП-12А |
| Количество заправочных головок | 12 |
| Эталонный выход | 3000 бутылок/час |
| Общая мощность | 1,12 кВт |
| Размеры | 1045×1050×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

Основная функция GFP-12A — обеспечить плавное стекание жидкости по внутренней стенке контейнера, обеспечивая постоянный уровень жидкости и подавляя пенообразование. В соответствии с логикой продукта эталонной станции, это, по сути, разливочная машина с низким вакуумом небольшой и средней производительности, подходящая для размещения после промывки и фильтрации и объединенная с укупоркой, усадкой крышек и маркировкой для формирования линии упаковки вина.

## Публичные функции

- Заполнение по уровню жидкости использует заполнение по разнице давлений жидкости, чтобы эффективно избежать проблем с пенообразованием при заполнении жидкостями, склонными к пенообразованию.
- Разбрызгивание вина, жидкость стекает вдоль внутренней стенки емкости, уменьшая пенообразование, вызванное ударом жидкости о дно бутылки.
- Заправочный клапан имеет точную конструкцию и небольшое количество уплотнений. Вся машина надежна и долговечна и подходит для непрерывного производства.
- Эластичная опорная конструкция для бутылок более щадящая на этапе подъема бутылок, что может уменьшить потерю стеклянных бутылок.
- Отсутствие розлива при отсутствии бутылки, а бутылки с дефектными горлышками не будут участвовать в розливе, что позволяет контролировать консистенцию готового продукта.
- Заправочный клапан имеет относительно простую конструкцию, что упрощает его разборку, очистку и обслуживание.

## Для каких проектов подходят

- Проекты по производству вина, фруктового вина и рисового вина мощностью около 3000 бутылок/час.
- Упаковка жидкостей в стеклянные бутылки, требующие постоянного уровня и внешнего вида жидкости.
- Сценарии заполнения, в которых соевый соус, уксус и т. д. также склонны к пенообразованию и уровень жидкости должен быть стабильным.
