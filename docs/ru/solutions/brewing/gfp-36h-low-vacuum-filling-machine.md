# GFP-36H разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-36H](/images/filling-products/gfp-36h-low-vacuum-filling-machine.webp)

«GFP-36H» — это модель с 36 головками из серии для наполнения с постоянным уровнем низкого вакуума. Он достиг уровня производительности 10 000 бутылок в час и подходит для более зрелых линий упаковки вин, рисового вина и жидких приправ для стеклянных бутылок.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-36H |
| Количество заправочных головок | 36 |
| Эталонный выход | 10 000 бутылок/час |
| Общая мощность | 3 кВт |
| Размеры | 2500×2350×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-36H соответствует требованиям низкого вакуумного наполнения и более высокой производительности. Он не только сохраняет преимущество фиксированного уровня жидкости, но также больше подходит для клиентов, у которых есть требования к посменной производительности и ритму. Он особенно подходит для автоматической закупорки, усадки крышек, маркировки и упаковки в картонные коробки для формирования непрерывной схемы производства.

## Публичные функции

- Низкая разница давления вакуума, подходит для стабильного наполнения легко вспенивающимися жидкостями.
- Жидкость стекает по стенке бутылки, что уменьшает образование пузырьков при ударе.
- Заправочный клапан имеет меньше уплотнений и более стабилен в долгосрочной перспективе.
- Держатель для бутылок имеет мягкую структуру, что помогает уменьшить потери стеклянных бутылок.
- Отсутствие розлива без бутылок, что улучшает стабильность качества готовой продукции на более поздних стадиях.
- Простота обслуживания и очистки, подходит для проектов массового производства.

## Для каких проектов подходят

- Крупномасштабная линия розлива вина или рисового вина производительностью около 10 000 бутылок/час.
- Высокочастотные проекты розлива, например, соевого соуса и уксуса в стеклянные бутылки.
- Клиенты, у которых есть требования к ритму наполнения, постоянству уровня жидкости и подключению внутренней автоматизации.
