# Укупорочная/усадочная машина

![Машина двойного назначения для прокатки и усадки крышек](/images/filling-products/cap-rolling-shrinking-machine.webp)

Машину двойного назначения для закатки и усадки колпачков можно использовать в качестве конкретной модели в серии машин для термоусадки резиновых колпачков. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия машин для термоусадки колпачков](/ru/solutions/cap-shrinking/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#cap-dual)
- Применимые сценарии: подходит для упаковки вина и подарочного масла с более прочным ощущением подарочной коробки.

## Сводка общедоступной информации

Эта серия термоусадочных машин в основном предназначена для термоусадочной герметизации термоусадочных резиновых крышек из ПВХ и специально используется для усадки крышек после укупоривания вина.
