# Серия машин фильтрации

![Фильтрация](/images/filling-solutions/filtering.png)

Фильтрационные машины используются для осветления жидкостей перед входом в секцию розлива. Они улучшают внешний вид продукта и стабильность процесса.

## Основные функции

- Удаление взвешенных частиц
- Повышение прозрачности и внешнего вида
- Снижение риска засорения голов и трубопроводов
- Подготовка более чистой жидкости к упаковке
