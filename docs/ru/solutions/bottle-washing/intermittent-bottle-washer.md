# Машина периодической промывки

![Машина для мойки бутылок прерывистого действия](/images/filling-products/intermittent-bottle-washer.webp)

На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия бутыломоечных машин](/ru/solutions/bottle-washing/)
