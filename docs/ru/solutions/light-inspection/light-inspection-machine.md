# Световая инспекционная машина

![Машина для светового контроля](/images/filling-products/light-inspection-machine.webp)

Машину для светового контроля можно использовать в качестве эталона для конкретных моделей серии машин для светового контроля. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия машин для светового контроля](/ru/solutions/light-inspection/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#light-standard)
- Применимые сценарии: подходит для прозрачных стеклянных бутылок и проектов по упаковке жидкостей с высокой добавленной стоимостью.

## Сводка общедоступной информации

Он реализует полностью автоматический метод проверки: вход в бутылку - зажим бутылки - поворот бутылки - взятие бутылки - удаление бутылки.

## Точки раскрытия информации

- Он реализует полностью автоматический метод проверки: вход в бутылку - зажим бутылки - поворот бутылки - взятие бутылки - удаление бутылки.
- Примените перевернутый контроль, который позволяет посторонним веществам в бутылке падать через перевернутый, чтобы улучшить качество контроля.
- Угол обзора большой, более 100°, что дает достаточно времени для проверки и сравнения с высокой точностью. После переворачивания бутылки внешняя кольцевая дорожка не загораживается, что делает линию обзора более четкой.
- Нажимные головки статического рычага и подвижного рычага подвижного патрона могут вращаться. Напорные головки мало изнашиваются и не повреждают бутылки или крышки.
- Светодиодный источник света, использующий светодиодный источник света, близкий к естественному свету, имеет низкое энергопотребление, высокую эффективность, не мерцает, не излучает, защищает глаза и не вызывает усталости при длительной работе.
- Нейлоновые губки обработаны методом профилирования и изготовлены по образцу неиспользованного горлышка бутылки. Зажим прочный, а крышку бутылки нелегко повредить.
