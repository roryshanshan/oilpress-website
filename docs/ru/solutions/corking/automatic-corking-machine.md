# Полностью автоматическая укупорочная машина

![Полностью автоматическая укупорочная машина](/images/filling-products/automatic-corking-machine.webp)

Полностью автоматическую укупорочную машину можно использовать в качестве эталона для конкретных моделей серии укупорочных машин. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия укупорочных машин](/ru/solutions/corking/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#cork-auto)
- Применимые сценарии: Подходит для непрерывных линий упаковки вина.

## Сводка общедоступной информации

Эта машина в основном подходит для укупорки вина и других стеклянных бутылок. В основном он подключается за разливочной машиной и используется для закупорки после наполнения.
