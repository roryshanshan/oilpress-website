# GFP-40H разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-40H](/images/filling-products/gfp-40h-low-vacuum-filling-machine.webp)

«GFP-40H» — это первоклассная основная модель в серии розлива с постоянным уровнем низкого вакуума, обеспечивающая производительность 12 000 бутылок в час, и подходит для проектов по производству высокотемпературного вина, рисового вина и производства соевого соуса и уксуса в стеклянных бутылках.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-40H |
| Количество заправочных головок | 40 |
| Справочная продукция | 12 000 бутылок/час |
| Общая мощность | 3,7 кВт |
| Габаритные размеры | 2500×2500×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-40H предназначен для заправочных станций фиксированного уровня с более высокими требованиями к сменному производству. Он подходит для клиентов, которые завершили планирование автоматизации предварительной обработки и последующей упаковки. Он используется для повышения эффективности наполнения легко пенящимися жидкостями до более высокого уровня, сохраняя при этом аккуратный и постоянный уровень жидкости в бутылке.

## Публичные функции

- Структура наполнения жидкостью с низким вакуумом, подходящая для стабильного наполнения безвоздушными жидкостями.
- Жидкость стекает по стене, что позволяет эффективно подавлять проблемы с пеной.
- Заправочный клапан надежен и долговечен, подходит для непрерывной высокочастотной работы.
- Эластичная конструкция для удержания бутылок может уменьшить повреждение бутылок.
- Без розлива без бутылок, что помогает контролировать консистенцию готовой продукции.
- Он относительно удобен для ежедневной разборки, чистки и обслуживания и подходит для полной совместимости линий.

## Для каких проектов подходят

- Проекты по производству вина, рисового вина и фруктовых вин мощностью около 12 000 бутылок/час.
- Проекты по производству соевого соуса и уксуса в стеклянных бутылках, требующие большего времени наполнения.
- Планируется подключение к средним и крупным комплексным линиям для автоматической укупорки, усадки крышек, маркировки, упаковки в коробки и паллетирования.
