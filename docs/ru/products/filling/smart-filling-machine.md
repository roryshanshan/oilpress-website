Интеллектуальная разливочная машина серии #

![Серия интеллектуальных разливочных машин](/images/filling-products/smart-filling-machine.webp)

Серия интеллектуальных разливочных машин в основном предназначена для автоматических линий розлива жидкостей в бутылки с учетом точности, скорости и возможности адаптации к нескольким бутылкам. Они подходят для производства со средними и высокими темпами при нормальном давлении, например, для пищевых масел и жидких приправ.

- Применимые сценарии: подходит в качестве основной машины для розлива масла в бутылки и жидкости для приправ.

## Сводка общедоступной информации

- Когда калиброванный объем составляет «500 мл», эталонная точность составляет «±1 мл».
- Публичная конфигурация охватывает заголовки `12/18/24/30/36`.
- Справочная скорость составляет 4000-13000 бутылок в час.

## Как выбрать

- «12-18 головок»: больше подходит для начинающих или средних проектов по розливу.
- «24-30 головок»: подходит для обычных основных производственных линий с учетом как скорости, так и требований к пространству.
- «36 головок»: непрерывная автоматическая линия, подходящая для более высокого темпа.
