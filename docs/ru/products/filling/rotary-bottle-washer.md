# Роторный ополаскиватель бутылок

![Вращающаяся машина для мытья бутылок](/images/filling-products/rotary-bottle-washer.webp)

Роторную промывочную машину можно использовать в качестве эталона для конкретных моделей серии промывочных машин. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия бутыломоечных машин](/ru/solutions/bottle-washing/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#bw-rotary)
- Применимые сценарии: подходит для непрерывного мытья обычных бутылочек.

## Сводка общедоступной информации

Он оснащен бутылями, обращенными вверх и вниз, шестиколесной трансмиссией, оснащен тремя распределителями воды, резервуаром для циркулирующей воды, водяным насосом, двумя промывками циркуляционной воды и одной промывкой чистой водой; его можно плавно регулировать и он имеет хороший эффект контроля воды.
