# GFP-48H разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-48H](/images/filling-products/gfp-48h-low-vacuum-filling-machine.webp)

«GFP-48H» — это высокопроизводительная модель из серии розлива с постоянным уровнем низкого вакуума, подходящая для проектов по производству винных, рисовых вин или приправ для стеклянных бутылок с производительностью 15 000 бутылок в час.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-48H |
| Количество заправочных головок | 48 |
| Справочная продукция | 15 000 бутылок/час |
| Общая мощность | 6 кВт |
| Габаритные размеры | 2800×2900×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-48H подходит для сценариев непрерывного наполнения с более высокой скоростью, уделяя особое внимание балансированию стабильности уровня жидкости, контроля пенообразования и более высокой скорости транспортировки. Если на более позднюю часть проекта запланированы автоматическая усадка крышек, маркировка, упаковка в картонные коробки и укладка на поддоны, этот уровень будет лучше соответствовать возможностям всей линии.

## Публичные функции

- Заполнение жидкостью при низком вакууме, подходит для жидкостей, склонных к пенообразованию.
- Жидкость стекает по стенкам бутылки, что позволяет уменьшить пенообразование при наполнении.
- Надежная конструкция, подходящая для высокоскоростного производства.
- Опора для бутылки мягкая, что снижает риск повреждения бутылки.
- Отсутствие розлива без бутылок, что повышает стабильность конечной линии.
- Легко разбирать, мыть и обслуживать, подходит для крупномасштабного производства.

## Для каких проектов подходят

- Высокопроизводительный винный проект с производственной мощностью около 15 000 бутылок в час.
- Полная линия по производству стеклянных бутылок с относительно полной конфигурацией автоматизации после упаковки.
- Клиенты, которым необходим компромисс между внешним видом уровня жидкости и производительностью.
