# Интеллектуальная машина для количественного розлива

![Интеллектуальная машина количественного наполнения](/images/filling-products/intelligent-quantitative-filling-machine.webp)

Интеллектуальная машина для количественного наполнения подчеркивает связь между наполнением и последующими действиями по укупорке, короблению и укупорке. Одна единица оборудования может выполнять несколько операций и подходит для автоматизированных линий, которые хотят уменьшить промежуточное ручное вмешательство и риск вторичного загрязнения.

## Позиционирование модели

- Серия: [Серия разливочных машин](/ru/solutions/filling/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#fill-dk)
- Применимые сценарии: подходит для автоматических линий, которые связывают подъем крышек, сортировку крышек, коробление крышек и запечатывание крышек после наполнения.

## Сводка общедоступной информации

- Компактная конструкция, подходящая для проектов модернизации автоматизации.
- Охватывает модели от «ДК-12» до «ДК-36».
- Справочная производительность `4000-13000 бутылок/час`

## Как выбрать

- «ДК-12/ДК-18»: подходит для пусковых автоматических линий или мелко- и среднесерийных проектов.
- «ДК-24/ДК-30»: подходит для стандартизированных основных производственных линий.
- «ДК-36»: подходит для проектов с более высокой производительностью и более полной автоматизацией процессов.
