# GFP-18A разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-18A](/images/filling-products/gfp-18a-low-vacuum-filling-machine.webp)

«GFP-18A» — это модель с 18 головками из серии для наполнения низкого вакуума и постоянного уровня. Он подходит для проектов по производству вин и ароматических жидкостей, которые требуют более высокой эффективности, чем модель с 12 головками, но не требуют временной производственной мощности более 24 головок.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-18А |
| Количество заправочных головок | 18 |
| Справочная продукция | 4500 бутылок/час |
| Общая мощность | 1,12 кВт |
| Габаритные размеры | 1500×1600×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-18A по-прежнему ориентирован на заполнение с фиксированным уровнем с целью сокращения времени цикла одной машины и одновременной стабилизации внешнего вида уровня жидкости. Для небольших и средних линий по производству вина, линий по производству фруктовых вин или линий по производству ароматизирующих жидкостей для стеклянных бутылок он часто используется в качестве первой переходной конфигурации при переходе с производительности с 3000 бутылок/час до 4500 бутылок/час.

## Публичные функции

- Структура наполнения жидкостью с низким вакуумом, подходящая для жидкостей, которые не содержат воздуха, но склонны к пенообразованию во время наполнения.
- Наполняющая жидкость стекает по стенкам бутылки, что улучшает контроль пенообразования.
- Заправочный клапан имеет компактную конструкцию и небольшое количество уплотнений, что делает его пригодным для непрерывного производства.
- Удержание бутылки более мягкое, что может помочь снизить вероятность потери бутылок.
- Отсутствие розлива без бутылок, что способствует повышению стабильности последующей упаковки.
- Техническое обслуживание и очистка более удобны, подходят для плановой замены и обслуживания линии.

## Для каких проектов подходят

- Проекты по производству вина, фруктового вина и рисового вина мощностью около 4500 бутылок в час.
- Клиенты, которые хотят найти более сбалансированный сегмент производственных мощностей от 12 до 24 голов.
- Проекты, требующие фильтрации, розлива, укупоривания и маркировки, которые необходимо объединить в линию по производству стеклянных бутылок малого и среднего размера.
