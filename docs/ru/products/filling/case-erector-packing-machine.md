# Автоматические машины для распаковки, формирования и картонирования коробок.

![Автоматическая машина для распаковки, формирования и упаковки картонных коробок](/images/filling-products/case-erector-packing-machine.webp)

Автоматические машины для распаковки, формирования и картонирования картонных коробок можно использовать в качестве эталона для конкретных моделей при картонировании и укладке на поддоны. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Упаковка, Паллетизация](/ru/solutions/packing-palletizing/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#pack-case)
- Применимые сценарии: подходит для интеграции автоматической распаковки и упаковки.

## Сводка общедоступной информации

Он автоматически подает сигнал тревоги при нехватке лент и коробок и использует управление экраном ПЛК +, что значительно облегчает работу и является незаменимым оборудованием для автоматизированного крупномасштабного производства;
