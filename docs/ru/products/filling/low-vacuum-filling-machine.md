# Низковакуумная разливочная машина

![Машина для розлива при низком вакууме](/images/filling-products/low-vacuum-filling-machine.webp)

Машину розлива с низким вакуумом можно использовать в качестве эталона для конкретных моделей серии пивоваренного оборудования. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия оборудования для пивоварения](/ru/solutions/brewing/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования] (/ru/products/filling-equipment#brew-vacuum)
- Применимые сценарии: подходит для легко пенящихся жидкостей, таких как вино, рисовое вино, соевый соус и уксус.

- Входная мощность: [`GFP-12A`](/ru/products/filling-equipment#brew-vacuum) / [`GFP-12B`](/ru/products/filling/gfp-12a-low-vacuum-filling-machine), эталонная производительность `3000 бутылок/час`

## Как выбрать модель?

- `GFP-12A / 12B`: больше подходит для малых и средних виноделен, пробных производственных линий и проектов по производству ароматизирующих жидкостей.
- `GFP-18A`: подходит для клиентов, которые хотят перейти с производительности 3000 бутылок в час, но на данный момент им не требуется более 24 головок.
- `GFP-24H / 30H`: это более распространенная основная модель линий среднего размера, которую легко связать с заглушкой, усадкой колпачка и маркировкой.
- `GFP-36H / 40H`: подходит для проектов непрерывного розлива стеклянных бутылок с производительностью 10 000–12 000 бутылок в час.
- `GFP-48H / 60H`: больше подходит для целых линейных проектов с более высокой степенью автоматизации серверной части.
