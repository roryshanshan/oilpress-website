# Линейная разливочная машина

![Линейная разливочная машина](/images/filling-products/linear-filling-machine.webp)

Линейная разливочная машина в основном используется для количественного розлива средних и больших бочек с жидкостями, подходит для упаковки в сегменте емкости 10-25 л и обычно используется в проектах по упаковке пищевого масла, химических жидкостей и крупных упаковочных жидкостей.

## Позиционирование модели

- Серия: [Серия разливочных машин](/ru/solutions/filling/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#fill-linear)
- Применимые сценарии: подходит для линейного наполнения средних и больших бочек, а также для проектов упаковки большой емкости.

## Сводка общедоступной информации

- Публичное покрытие модели `ZGP-4/ZGP-6/ZGP-8/ZGP-10`
- Диапазон наполнения `10-25 л`
- Эталонная точность `≤0,2%`

## Как выбрать

- «ZGP-4/ZGP-6»: подходит для проектов с большими стволами средней и низкой производительности.
- «ZGP-8»: подходит для основной лески среднего размера.
- «ZGP-10»: подходит для непрерывного наполнения чанов с более длительным циклом.
