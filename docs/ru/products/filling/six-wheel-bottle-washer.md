# Шестиколесная промывочная машина

![Шестиколесная-мойка для бутылок](/images/filling-products/six-wheel-bottle-washer.webp)

Шестиколесную промывочную машину можно использовать в качестве образца для конкретных моделей серии промывочных машин. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия бутыломоечных машин](/ru/solutions/bottle-washing/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#bw-six-wheel)
- Применимые сценарии: больше подходит для мытья стеклянных и ПЭТ-бутылок на средней скорости.

## Сводка общедоступной информации

Эта машина использует принцип цепной передачи с роликовыми втулками. Вся конвейерная цепь имеет точную передачу и хорошую износостойкость. Для оборотной воды используется следящее распыление и распыление по фиксированной трубе. В части очистки воды используется распределитель воды для отслеживания и распыления, чтобы сэкономить источники воды и улучшить ее использование.
