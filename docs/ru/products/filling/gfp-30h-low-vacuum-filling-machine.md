# GFP-30H разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-30H](/images/filling-products/gfp-30h-low-vacuum-filling-machine.webp)

«GFP-30H» — это модель с 30 головками из серии для наполнения с постоянным уровнем низкого вакуума. Он подходит для производства вин, фруктовых вин, рисовых вин или жидких приправ для стеклянных бутылок, которые вступили в стадию непрерывного производства.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-30H |
| Количество заправочных головок | 30 |
| Эталонный выход | 7000 бутылок/час |
| Общая мощность | 2,5 кВт |
| Габаритные размеры | 2100×2170×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-30H предназначен для улучшения возможностей заполнения жидкостью при низком уровне вакуума до более стабильного диапазона средней и высокой производительности, сохраняя при этом совместимость с жидкостями, склонными к пенообразованию. Часто это очень практичный уровень для покупателей вина, которым нужна большая эффективность и красивый и стабильный уровень жидкости.

## Публичные функции

- Заполнение жидкостью при низком вакууме, подходит для заполнения безгазовых жидкостей на фиксированном уровне.
- Наполняющая жидкость стекает по стенке, что позволяет уменьшить пенообразование в бутылке.
- Заправочный клапан имеет надежную конструкцию и пригоден для непрерывной работы в течение длительного времени.
- Эластичная опорная конструкция для бутылок более удобна для стеклянных бутылок.
- Никакого розлива без бутылок, что помогает стабилизировать ритм упаковки на более позднем этапе.
- Удобнее разбирать, мыть и обслуживать, что удобно для ежедневной смены и чистки.

## Для каких проектов подходят

- Линия упаковки вина, фруктовых вин и рисового вина производительностью около 7000 бутылок/час.
- Проекты по производству стеклянных бутылок среднего и большого размера, требующие укупорки, усадки крышек, маркировки и упаковки.
- Клиенты, которые хотят продолжить наращивание производственных мощностей по сравнению с моделью с 24 головками, но еще не достигли конфигурации с 36 и более головками.
