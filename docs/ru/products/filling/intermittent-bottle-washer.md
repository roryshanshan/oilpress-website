# Машина периодической промывки

![Машина для мойки бутылок прерывистого действия](/images/filling-products/intermittent-bottle-washer.webp)

Ополаскиватель периодического действия можно использовать в качестве эталона для конкретных моделей серии ополаскивателей. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия бутыломоечных машин](/ru/solutions/bottle-washing/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#bw-intermittent)
- Применимые сценарии: Подходит в качестве промывочной установки начального уровня или средней или низкой скорости.
