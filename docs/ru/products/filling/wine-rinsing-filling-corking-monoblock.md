# Комбинированная машина для ополаскивания, розлива и укупорки вина.

![Комбинированная машина для ополаскивания, розлива и укупорки вина](/images/filling-products/wine-rinsing-filling-corking-monoblock.webp)

Комбинированную машину для ополаскивания, розлива и укупорки вина можно использовать в качестве эталона для конкретных моделей серии винодельческого оборудования. На этой странице упорядочено расположение моделей, информация о параметрах и вспомогательные предложения, основанные на общедоступной информации о продукте, для облегчения предварительного выбора и перехода по страницам.

## Позиционирование модели

- Серия: [Серия пивоваренного оборудования](/ru/solutions/brewing/)
- Вход в обзор параметров: [Библиотека параметров фасовочно-упаковочного оборудования](/ru/products/filling-equipment#brew-triple)
- Применимые сценарии: Подходит для всей линейки вин и фруктовых вин.

## Сводка общедоступной информации

Эта тройная машина в основном предназначена для розлива вина и фруктовых вин. Это оборудование органично сочетает в себе ополаскиватель с переворачивающимся устройством, машину для розлива при отрицательном давлении и укупорочную машину. Оборудование имеет компактную конструкцию и экономит место для установки.
