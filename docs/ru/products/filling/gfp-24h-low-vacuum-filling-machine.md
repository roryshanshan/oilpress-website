# GFP-24H разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-24H](/images/filling-products/gfp-24h-low-vacuum-filling-machine.webp)

«GFP-24H» — это модель с 24 головками из серии для розлива с постоянным уровнем низкого вакуума, а ее производственная мощность вошла в общий диапазон линий по упаковке вина среднего размера. Это относительно распространенная основная модель для производства вина, фруктового вина, рисового вина, а также проектов по производству соевого соуса и уксуса в стеклянных бутылках.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-24H |
| Количество заправочных головок | 24 |
| Эталонный выход | 6000 бутылок/час |
| Общая мощность | 1,85 кВт |
| Габаритные размеры | 1900×1900×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-24H в основном удовлетворяет сбалансированные потребности проектов средней мощности в обеспечении постоянства уровня жидкости, контроля пенообразования и непрерывного производственного ритма. Его можно использовать как автономную машину, а также он подходит для подключения к вспомогательному оборудованию, такому как ополаскивание бутылок, укупорка бутылок, усадка крышек и маркировка, чтобы сформировать полную линию розлива и упаковки стеклянных бутылок.

## Публичные функции

- Метод наполнения жидкостью при низком вакууме позволяет эффективно контролировать состояние наполнения жидкостей, склонных к пенообразованию.
- Жидкость стекает по стенкам бутылки, уменьшая пенообразование на дне.
- Заправочный клапан имеет простую конструкцию, его легче разбирать и чистить.
- Опорная конструкция для бутылок мягкая и подходит для транспортировки и подъема стеклянных бутылок.
- Отсутствие наполнения без бутылок, что может снизить риск пустого наполнения.
- Подходит для стандартизированного розлива вина, фруктового вина, рисового вина и других продуктов из стеклянных бутылок.

## Для каких проектов подходят

- Линия по упаковке вин и фруктовых вин среднего размера производительностью около 6000 бутылок/час.
- Линии ароматизированных жидкостей в стеклянных бутылках, требующие стабильного уровня жидкости и лучшего контроля пенообразования.
- Есть надежда, что в будущем будет добавлен целый проект линии автоматической укупорки, усадки крышек, маркировки и упаковки в картонные коробки.
