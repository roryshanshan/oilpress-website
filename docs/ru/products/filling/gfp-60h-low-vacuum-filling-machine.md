# GFP-60H разливочная машина низкого вакуума

![Машина для розлива с низким вакуумом GFP-60H](/images/filling-products/gfp-60h-low-vacuum-filling-machine.webp)

«GFP-60H» — это более производительная версия серии устройств для розлива с постоянным уровнем низкого вакуума, предназначенная для крупномасштабных проектов по наполнению стеклянных бутылок жидкостью со скоростью 20 000 бутылок в час.

## Основные параметры

| Проект | Параметры |
| --- | --- |
| Модель | GFP-60H |
| Количество заправочных головок | 60 |
| Эталонный выход | 20 000 бутылок/час |
| Общая мощность | 7,5 кВт |
| Габаритные размеры | 3500×3500×2200 мм |
| Метод заполнения | Заполнение низкого вакуума/постоянного уровня |
| Применимые жидкости | Вино, рисовое вино, соевый соус, уксус и другие легко пенящиеся жидкости |

## Что делает это устройство?

GFP-60H в основном служит для крупномасштабного наполнения при низком вакууме и постоянном уровне. Для клиентов, у которых уже есть основа автоматизации предварительной обработки, фильтрации, транспортировки и последующей упаковки, этот уровень больше подходит в качестве полной линии наполнения.

## Публичные функции

- Раствор для наполнения жидкостью при низком вакууме, подходит для жидкостей, склонных к пенообразованию.
- Жидкость прилипает к стенке и стекает вниз, что позволяет уменьшить образование пузырьков на дне.
- Надежная конструкция, подходящая для непрерывного высокоинтенсивного производства.
- Мягкое удерживание бутылки помогает защитить стеклянную бутылку.
- Ни бутылки, ни наполнения, легко контролировать ритм всей линии.
- Простота обслуживания и очистки, подходит для ежедневного обслуживания крупномасштабных проектов.

## Для каких проектов подходят

- Полная линия для ароматизированных жидкостей в винных или стеклянных бутылках производительностью 20 000 бутылок/час.
- Клиенты, у которых есть более высокие требования к автоматизации внутренней упаковки, паллетирования и отслеживания.
- Заводы, которые хотят выполнять крупносерийные заказы на одной и той же платформе розлива.
